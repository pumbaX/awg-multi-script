# AWG Multi Script

Универсальный менеджер **AmneziaWG** для быстрого развёртывания VPN на VPS одной командой.

## Быстрый старт

```bash
curl -s https://raw.githubusercontent.com/pumbaX/awg-multi-script/main/awg.sh -o /tmp/awg.sh
sudo bash /tmp/awg.sh
```

## Описание

**AWG Multi Script** — интерактивный bash-менеджер для установки, настройки и управления AmneziaWG VPN-сервером. Поддерживает все версии протокола AmneziaWG от 1.0 до 2.0, автоматически генерирует обфускационные параметры и позволяет управлять клиентами без ручного редактирования конфигов.

## Возможности

- **Один скрипт** — установка, генерация, управление клиентами в едином интерфейсе
- **Все версии протокола** — WireGuard, AWG 1.0, AWG 1.5, AWG 2.0 на выбор
- **Автогенерация параметров обфускации** — Jc, Jmin, Jmax, S1-S4, H1-H4 диапазоны
- **I1 имитация трафика** — Google DNS, Яндекс DNS или реальный QUIC-пакет через API
- **Управление клиентами** — добавление, просмотр статистики, QR-коды
- **Автоопределение версии** — при добавлении клиента версия читается из конфига сервера
- **NAT + FORWARD** — автоматическая настройка iptables с сохранением через hook
- **UFW** — автоматическое открытие портов с защитой SSH
- **Статистика** — отображение трафика клиентов (↑ выгрузка / ↓ загрузка) в МБ
- **Backup** — автоматический бэкап конфига перед пересозданием сервера
- **Полное удаление** — очистка пакетов, конфигов, правил firewall одной командой

## Требования

- Ubuntu 22.04 / 24.04
- KVM-виртуализация (не OpenVZ/LXC)
- Публичный IPv4
- Root-доступ

## I1 — имитация трафика

| Вариант | Описание |
|---|---|
| Google DNS | Статический DNS-пакет, совместим со всеми версиями AmneziaVPN |
| Яндекс DNS | Статический DNS-пакет Кинопоиска/Яндекса |
| QUIC API | Реальный QUIC Initial packet по домену — максимальная маскировка |
| Без имитации | Базовый режим AWG |

## Импорт на клиенте

Используй **AmneziaVPN**:
- Сканирование QR-кода из терминала
- Импорт файла `.conf` через «Добавить туннель → Из файла»

Скачать: [amnezia.org/downloads](https://amnezia.org/downloads)

## Файлы

```
/etc/amnezia/amneziawg/awg0.conf   — серверный конфиг
/root/client1_awg2.conf            — первый клиент
/root/<name>_awg2.conf             — дополнительные клиенты
```
